Blooming Head

Medium: Bamboo Skewers & Tissue Paper

Date: 2021.11

This project was about to create a 3D-formed structure by using bamboo skwers which could interact with human body.

I want my structure to have more natural forms, and when looking at the garden outside the classroom, I finally decided to create a flower-shaped hat to add some vividness to the freezing winter of Ann Arbor.

Loong

Medium: Paper, Micron Pen

Date: 2021.12

This project was about showing different textures through drawing different patterns.

I thought it would be ordinary if I only use different geometric shapes to contain each texture.

As I am always intending to combine Chinese elements and style into my creation, I finally decided to present a Chinese Loong holding a piece of scoll containing the four typical plants of China: Plum, Orchid, Bamboo and Chrysanthemum.

Me: Color

Medium: Paper, Ink, Paints

Date: 2021.11

This project is about to explore and play more with the grey scale and color.

The first version shows the grey scale by not using grey colors, but using different patterns with different density to show the difference between light and dark.

The second version aims at totally change the color scheme of the picture which adds more dramatic effect to the image.

Lantern

Medium: Taskboard, Tissue Paper, Plastic Paper

Date: 2021.12-2022.03

This project was my first experiment with the Laser Cut Machine, and I created two versions of it.

For the first version, I was only thinking about creating a Chinese styled lattern and incorporating Chinese traditional patterns. But latter, I started to think of my own identity as an international student, and I want to relate my own experience with my works.

As the result, I replace the repetitive patterns with colorful shapes which refer to the Tiffany glasses. This also added more color to the light and made the final effect much prettier.
